# Mac-Changer
This hacker tool i created allows users to change their mac addresses logically and less than a second. Enjoy the tool :)

## Usage : 
### First step
-> Download the tool copying .git url of the repo and typing `git clone <url>` on your linux terminal

### Second step
->  change your directory to the repo dir

### Third step
-> to use the tool, run the code given below

`python MacChanger.py -i <your interface> --mac <the mac address you want to change your old mac address to>`
